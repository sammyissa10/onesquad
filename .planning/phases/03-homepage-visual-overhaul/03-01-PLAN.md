<task type="auto">
  <name>Task 2: Rebuild Hero as asymmetric bento grid with oversized typography</name>
  <files>src/components/sections/Hero.tsx</files>
  <action>
Completely rewrite Hero.tsx as an asymmetric bento grid layout inspired by Kota.co.uk. The hero must feel hand-crafted, not templated.

**Layout structure (CSS Grid, NOT flexbox):**
- Full viewport: `min-h-[100vh] min-h-dvh` (dvh with vh fallback per research)
- Background: `bg-navy` (dark hero, per section rhythm decision)
- Grid: Use `grid grid-cols-4 md:grid-cols-6` with explicit `col-span` and `row-span` (NO grid-auto-flow: dense)
- Padding: `p-4 md:p-6 lg:p-8` outer, with `gap-3 md:gap-4` between blocks

**Bento blocks (desktop 6-col grid):**
1. **Headline block** (col-span-3, row-span-2): Oversized headline using Coolvetica. Text should be `text-5xl md:text-7xl lg:text-[6rem] xl:text-display font-black leading-[0.9]`. Headline: a bold capability statement like "We Build Digital Empires For Small Businesses" (confident, not generic). Include subtext in `text-xl text-white/70` and dual CTAs (primary coral "See Our Plans" linking to /pricing + outline "Get a Free Quote" linking to /contact). Use MagneticButton wrapping each CTA. Add `data-cursor="button"` on CTA links.
2. **Visual accent block** (col-span-3, row-span-2): Large block with `bg-gradient-to-br from-coral to-peach` and rounded-3xl. Inside, place the brand slogan "Unlock your digital potential" in large text as a visual element, with a subtle pattern or the logo icon. Add `data-cursor="card"`.
3. **Stat block** (col-span-2, row-span-1): Dark translucent block `bg-white/5 backdrop-blur-sm rounded-2xl` showing a key stat (e.g., "150+ Projects Delivered"). Use the stats from constants.ts — pick the most impactful one.
4. **Logo/brand block** (col-span-1, row-span-1): Small block `bg-coral/20 rounded-2xl` centered, showing the brand name or a small visual element.
5. **Quick CTA block** (col-span-1, row-span-1): Small block `bg-peach text-navy rounded-2xl font-bold` with a quick action like "Start Now" with an arrow.
6. **Trust block** (col-span-2, row-span-1): Block `bg-blue/20 rounded-2xl` showing trust indicator (e.g., "Trusted by 50+ Businesses" or "No Long-term Contracts").

**Mobile (4-col grid):**
All blocks become col-span-4 (full width) except stat/logo/CTA blocks which can be col-span-2 for a 2-column sub-grid. The headline block should still be prominent. Stack vertically with the headline first.

**Entrance animation (Framer Motion):**
- Wrap content in `<MotionConfig reducedMotion="user">` for accessibility
- Use staggerChildren (0.12s) on the grid container with delayChildren (0.15s)
- Each bento block fades in with slight y-offset: `{ opacity: 0, y: 20 }` -> `{ opacity: 1, y: 0 }`
- Custom easing: `[0.22, 1, 0.36, 1]` for smooth deceleration
- Duration per item: 0.5s

**Copy tone:** Bold capability positioning. NOT "Two Teams. One Mission." (that's the old hero). Write new headline that declares what OneSquad does confidently. Keep the brand slogan "Unlock your digital potential" as supporting text.

**Technical notes:**
- Keep "use client" directive
- Import stats and siteConfig from @/lib/constants
- Import Button from @/components/ui/Button
- Import MagneticButton from @/components/ui/MagneticButton
- Import motion, MotionConfig from framer-motion
- Import ArrowRight from lucide-react
- Import Link from next/link
- Do NOT use the Container component for the hero — the bento grid should go edge-to-edge within the hero section
- The hero is the first section visitors see — it must make a dramatic first impression
